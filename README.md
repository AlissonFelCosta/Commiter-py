# Commiter-py

Resumindo
 - Abre o arquivo codColab.txt e cola no colab
 - Alterar a variável SEU_NOME para o usuário do git
 - Alterar a variável SEU_EMAIL para o email do git
 - Precisa ter criado o repositório
 - Copiar o link do repositorio ex: REPO_DESTINO = "https://github.com/AlissonFelCosta/Comitter-py"
 - Criar a chave TOKEN do git, substituir na URL REPO_DESTINO = "https://SUACHAVETOKENAQUI@github.com/AlissonFelCosta/Comitter-py"
 - Rodar o código no Colab
